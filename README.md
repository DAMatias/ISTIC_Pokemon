# ISTIC Pokémon - Grupo 1

Proyecto desarrollado por el **Grupo 1: Díaz y Somma**.

## 📖 Descripción del Proyecto
Esta aplicación es un catálogo interactivo de Pokémon que permite a los usuarios visualizar una lista completa de personajes, consultar sus estadísticas detalladas, tipos y gestionar una lista de favoritos. La aplicación está diseñada para ofrecer una experiencia fluida de búsqueda y exploración de la información oficial del mundo Pokémon.

## 🚀 API Utilizada
El proyecto consume datos de la **[PokeAPI](https://pokeapi.co)**, una API RESTful gratuita y pública que proporciona una base de datos detallada de todos los Pokémon, sus habilidades, estadísticas y tipos.

## 🛠️ Requisitos para ejecutar
* **Android Studio:** Versión Iguana o superior (recomendado).
* **SDK Mínimo:** API 24 (Android 7.0).
* **Conexión a Internet:** Requerida para la carga de datos desde la API.

## 📱 Características principales
* **Listado de Pokémon:** Visualización dinámica con carga eficiente.
* **Detalle avanzado:** Pantalla informativa con imágenes y barras de progreso para estadísticas (HP, Ataque, Defensa, Velocidad).
* **Buscador:** Filtrado en tiempo real por nombre de Pokémon.
* **Favoritos:** Funcionalidad para marcar y gestionar tus Pokémon preferidos, persistiendo la sesión mediante DataStore.
